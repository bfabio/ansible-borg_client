Introduction
============

The ``debops.borg_client`` Ansible role can be used to manage backups with :program:`borg`.

Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.0.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops.borg_client
